# Local Development

## Use Ganache
Instead of using testnet, we can using local node like ganache. It makes SC deployment become more easy.

## Use truffle dashboard
The alternative way using testnet without providing our Private key is by using truffle dashboard

Install truffle dashboard 
```bash
npm install -g truffle
truflle dashboard
```
